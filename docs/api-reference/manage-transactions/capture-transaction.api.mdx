User can capture a delay Capture transaction early earlier then the set day. Authenticate this request using HTTP Basic Auth by adding a bearer token to the header, e.g. Authentication: Bearer {{access_token}}. You can get the {{access_token}} from the create token request.

The previous transaction must have delay_capture_days set to be greater than 0 in the intent.
